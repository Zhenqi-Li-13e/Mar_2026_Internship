# 宽基指数择时策略

基于动量和效率系数的ETF轮动策略，对宽基指数进行择时轮动。

---

## 策略逻辑

### 核心指标

- **动量** = 100 × ln(价格中枢20日前 / 价格中枢当前)
- **价格中枢** = (开盘价 + 最高价 + 最低价 + 收盘价) / 4
- **效率系数** = 方向性 / 波动性
- **方向性** = |窗口期净涨跌|
- **波动性** = Σ|每日对数收益率|
- **评分** = 动量 × 效率系数

### 选股规则

- 每天选择评分最高的 **Top 2 ETF**
- 每只ETF分配 **50% 仓位**（满仓）

### Look Ahead Bias 防护

- 动量计算使用 `shift(20)` 获取20日前价格
- 效率系数使用 `shift(1)` 获取前一日数据
- 使用前一日评分决定今日仓位

### 交易成本

| ETF类型 | 手续费 | 滑点 | 单边成本 |
|---------|--------|------|----------|
| A股ETF  | 0.01%  | 0.02% | 0.03%   |
| 跨境ETF | 0.02%  | 0.02% | 0.04%   |

> 买入/卖出各扣一次

### ETF池（10只）

- **A股**：上证50、沪深300、中证500、中证1000、创业板、科创50、黄金ETF
- **跨境**：纳指100、标普500
